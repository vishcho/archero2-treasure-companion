# Archero 2 Treasure Companion

Companion to help you track treasures on the treasure map in Archero 2.

## Features

- 10x12 grid treasure map
- Drag and drop treasure pieces onto the map
- Rotate treasure pieces
- Remove placed treasures
- Visual feedback for valid/invalid placements

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000/archero2-treasure-companion](http://localhost:3000/archero2-treasure-companion) in your browser.

### Build

```bash
npm run build
```

## Usage

1. Drag treasure pieces from the right panel onto the 10x12 grid
2. Click the rotate button to rotate a placed piece
3. Click the trash button to remove a placed piece
4. Use "Reset All" to clear the entire map
